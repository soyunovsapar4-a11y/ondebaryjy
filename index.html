Активность опылителей и сроки обработки

Конец марта – апрель: Ранние опылители

Bombus spp. и Osmia spp. начинают лёт при t ≥5–8°C. Первыми опыляют абрикос, сливу, грушу, яблоню. 🚫 Обработки пестицидами строго запрещены.

Апрель – май: Пик активности

Apis mellifera (≥10–12°C) + Andrena spp. Массовое цветение. Рекомендуемая плотность: 2–4 семьи/га для яблони. ✅ Обработки только ночью.

Май – июнь: Персик, нектарин, черешня

Apis продолжает активную работу. Осторожно с системными инсектицидами — остатки в нектаре и пыльце.

Июнь – август: Летний период

Lasioglossum и Halictus активны (персик, поздний абрикос). Основные обработки фунгицидами. Неоникотиноиды системные — опасны.

Август – октябрь: Завершение сезона

Пчёлы готовятся к зимовке. Остатки пестицидов в воске влияют на зимостойкость. Гербициды под урожай.

🧪 Классы пестицидов

Применяемые в теплицах и на открытом грунте Центральной Азии

⚠️
Основные классы пестицидов ЦА — системные неоникотиноиды, пиретроиды, ФОС/карбаматы, стробилуриновые и азольные фунгициды, глифосатные гербициды, акарициды. Неоникотиноиды представляют наибольшую угрозу для опылителей.

🔴 Неоникотиноиды — наиболее опасный класс для опылителей

Системные инсектициды. Блокируют никотин-ацетилхолиновые рецепторы (nAChR). Обнаруживаются в нектаре (до 15 мкг/кг) и пыльце (до 100 мкг/кг). Клотианидин сохраняется в почве до 19 лет.

🟠 Пиретроиды — контактно-кинетические токсины

Воздействуют на потенциал-зависимые натриевые каналы нейронов. Высокотоксичны при контакте, но быстро деградируют под УФ. В теплицах (защита от УФ) период действия значительно увеличивается.

🔵 Фунгициды — косвенные и сублетальные эффекты

Прямая токсичность обычно низкая, но азолы и стробилурины подавляют детоксикационные ферменты CYP450 пчёл, усиливая токсичность инсектицидов в 5–1141 раз.

⚠️
Синергизм! Проциназол + имидаклоприд: LC50 снижается в 35 раз. Никогда не смешивайте азолы с неоникотиноидами.

🟢 Гербициды — прямой риск невысокий, но есть нюансы

Специфически ингибируют ферменты растений. Главные угрозы — уничтожение кормовой базы и воздействие глифосата на кишечную микробиоту пчёл (Motta et al., 2018, PNAS).

🐝 Опылители

Видовой состав, температурные пороги, сезонный календарь Центральной Азии

🍯

Apis mellifera — Медоносная пчела

Активность: Апрель – Октябрь · Мин. t: ≥10–12°C · Оптимум: 20–30°C. Основной промышленный опылитель. Опыление увеличивает массу яблок на +11–26% и число завязей в 5.6×. Рекомендуемая плотность: 2–8 семей/га.

УправляемаяЧувствительна к неоник.
🐝

Bombus spp. — Шмели

Активность: Конец марта – Сентябрь · Мин. t: ≥5–8°C · Оптимум: ~15–20°C. Незаменимы при низких температурах. Добавление B. terrestris к Apis повышает завязываемость яблок на +15%. Используют «баззинг» (вибрационное опыление).

ХолодостойкиеТеплицы
🏗️

Osmia spp. — Пчёлы-каменщицы

Активность: Март – Июнь · Мин. t: ≥10°C. Лучший опылитель яблони и вишни в плохую погоду. Одна осмия заменяет ~120 медоносных пчёл по эффективности опыления яблони. Гнездятся в трубках и стеблях.

×120 эффективность
🌍

Andrena spp. — Земляные пчёлы

Активность: Апрель – Май · Мин. t: ≥10–12°C. Специализированные весенние опылители. >100 видов в регионе ЦА. Гнездятся в земле — уязвимы к почвенным пестицидам (хлорпирифос, системные неоникотиноиды).

Уязвимы к ФОС
✨

Lasioglossum / Halictus — Галиктиды

Активность: Апрель – Сентябрь · Летний пик. Важны для опыления персика и позднего абрикоса. Маленький размер позволяет опылять цветки с узкими венчиками. ~1000 видов в мире.

⚠️ Риски и токсичность

Летальные и сублетальные эффекты пестицидов на опылителей

🚨
По данным FAO и IPBES, численность диких опылителей снижается на 30–70% в сельхозрегионах. Неоникотиноиды — главная причина CCD (коллапс колоний). В 2018 году ЕС запретил три основных неоникотиноида на открытом воздухе. В ЦА аналогичных ограничений нет.
🌸

Нектар и пыльца

Системные пестициды: нектар до 15 мкг/кг, пыльца до 100 мкг/кг. Хроническое отравление всей колонии через перга и мёд.

💧

Гуттация

Имидаклоприд в гуттации кукурузы: 10–1000 мкг/л. В 10–100 раз больше, чем в почвенной воде. Критически опасна ранним утром.

💨

Пылевые частицы

Протравленные семена при посеве образуют пылевое облако. LD50 превышается при контакте с 1–5 частицами пыли.

🌱

Почвенный путь

Andrena гнездятся в почве. Клотианидин: 148–6931 дней в почве. Имидаклоприд: 48–190 дней.

🏗️

Гнездовые материалы

Неоникотиноиды накапливаются в воске (до 20 ppb). Личинки подвергаются воздействию всё развитие → CCD.

🌿

Прямой контакт

Дельтаметрин на листьях: 5–7 дней. В теплице: до 14 дней. Острая гибель в течение часов при прямой обработке.

Сравнение LD50 для Apis mellifera

*Меньшее LD50 = большая токсичность. Оральный путь воздействия.

Сублетальные и системные эффекты

🧠 Нарушение навигации и обучения+
  • Снижение визитов к цветкам на 40%
  • Потеря ориентации при возврате в улей (+9% потерянных пчёл с GPS)
  • PER-обучение снижено на 60%
  • Вербальные танцы менее точны
👑 Репродуктивные нарушения шмелей+
  • Имидаклоприд 6 ppb: снижение веса гнезда B.terrestris на 85%
  • Тиаметоксам: снижение возвращаемости маток в 2.3 раза
  • Уменьшение числа половых особей в колонии
🔥 CCD — Коллапс медоносных колоний+

Рабочие пчёлы исчезают, оставляя матку и расплод. Причины:

  • Нарушение навигации → пчёлы не возвращаются
  • Иммуносупрессия → Nosema + вирусы в ослабленных колониях
  • Накопление в воске → хроническое отравление расплода
  • ЕС запретил имидаклоприд, клотианидин, тиаметоксам (2018)

Пошаговое руководство по индексации сайта

⭐
Это полное пошаговое руководство для вашего GitHub Pages сайта. Следуйте шагам — и ваш сайт появится в Google, Yandex и Bing в течение 1–4 недель.

📦 Шаг 1: Деплой на GitHub Pages

1

Создайте репозиторий на GitHub

Перейдите на github.com → кнопка New → назовите репозиторий your-name.github.io (для главного домена) или любое имя (для subdirectory).

2

Загрузите файлы

Вариант A — через браузер: нажмите Add file → Upload files и перетащите все файлы проекта.
Вариант B — Git командная строка:

git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/ВАШ_НИК/ВАШ_РЕПО.git
git push -u origin main
3

Включите GitHub Pages

Перейдите в репозиторий → Settings → Pages → Section "Source" → Deploy from a branch → ветка main → папка / (root) → нажмите Save.
Через 1–5 минут сайт будет доступен по адресу https://ВАШ_НИК.github.io/ВАШ_РЕПО/

🔍 Шаг 2: Google Search Console

4

Перейдите в Google Search Console

Откройте search.google.com/search-console → войдите с аккаунтом Google → нажмите «Добавить ресурс».

5

Выберите тип ресурса

Выберите «URL-prefix» (не "Domain") и введите полный URL вашего сайта:
https://ВАШ_НИК.github.io/ВАШ_РЕПО/

6

Верификация через HTML-тег

Выберите метод «HTML tag». Скопируйте код вида:
<meta name="google-site-verification" content="ВАШИ_СИМВОЛЫ" />

Откройте файл index.html → найдите строку с ВСТАВЬТЕ_ВАШ_VERIFICATION_CODE_ЗДЕСЬ → замените на ваш код → сохраните → задеплойте (git push). Затем вернитесь в Search Console и нажмите «Подтвердить».

7

Отправьте Sitemap

После верификации: в левом меню → Sitemaps → в поле вставьте:
https://ВАШ_НИК.github.io/ВАШ_РЕПО/sitemap.xml
→ нажмите «Отправить». Статус должен стать «Успешно».

8

Запросите индексацию

В поле URL Inspection вверху введите URL главной страницы → нажмите «Запросить индексирование». Первые страницы появятся в Google через 3–14 дней.
